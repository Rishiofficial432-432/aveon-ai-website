## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rishiofficial432-432/aveon-ai-website.git
   ```

2. **Install dependencies**
   ```bash
   cd aveon-ai-website
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **Frontend Framework**: React.js
- **Styling**: Tailwind CSS
